# 📈 Polynomial Regression on Job Level Dataset

Predicting salaries using Polynomial Regression in Python

---

## 📖 Overview

This repository demonstrates how to build a **Polynomial Regression** model using a custom **Job Level** dataset.

The notebook covers the complete machine learning workflow, from data preprocessing and polynomial feature generation to model training, prediction, and visualization using Python and Scikit-Learn.

It serves as a beginner-friendly yet practical example for understanding nonlinear regression in supervised machine learning.

---

## 🎯 Project Objectives

- Explore the Job Level dataset
- Perform data preprocessing
- Generate polynomial features
- Train a Polynomial Regression model
- Predict salaries for different job levels
- Visualize the regression curve
- Evaluate model performance

## 📊 Dataset

The custom **Job Level** dataset contains **1,100 samples** with the following features:

| Feature | Description |
|----------|-------------|
| Job Level | Employee job level |
| Salary | Target variable |

The goal is to predict **Salary** using Polynomial Regression based on the available features.
